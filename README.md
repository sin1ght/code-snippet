## code-snippet
> 一个花里胡哨的代码片段管理与分享平台

#### 功能实现:
- 用户登录注册
- 分类管理
- 片段管理、分享，编辑
- markdown支持，实时预览，tab与shift tab支持
- 可用路由 /login 和 /home

#### 运行:
client
- cd client
- npm install
- npm start

server（基于cors跨域，限制了origin，http://localhost:3000，可在app.ts中更改）
- cd server
- npm install
- npm run db_init 初始化表
- npm start
